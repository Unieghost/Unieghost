## Featured Project

### Project 001 – Operation Blackout

**Purple Team Active Directory Intrusion Simulation**

Operation Blackout is a full Purple Team campaign designed to simulate and detect a multi-stage intrusion chain inside a controlled Active Directory lab.

The project covers execution, discovery, persistence, account manipulation, privilege escalation and PsExec-style lateral movement.

**Highlights:**

* 9 attack phases fully executed and documented
* Wazuh, Sysmon and Windows Event Logs used for detection
* MITRE ATT&CK mapping across all phases
* Domain Controller monitoring with account and privilege abuse detection
* Final report, detection matrix, IOC documentation and lessons learned
* Screenshots and evidence organized for SOC-style investigation workflow

**Main techniques covered:**

* PowerShell Execution
* Network Service Discovery
* Scheduled Task Persistence
* Registry Run Key Persistence
* Windows Service Creation
* Domain User Creation
* Domain Admins Group Modification
* PsExec-style Remote Execution
